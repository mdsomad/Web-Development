### Perspective Properties Use view
```sh
.container {
  width: 100vw;
  height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  box-shadow:  inset 0 0 0 5px #4861ec;
  perspective: 1000px;
}
```
